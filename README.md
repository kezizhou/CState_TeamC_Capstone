# CState_TeamC_Capstone
A near miss incident reporting tool for the manufacturing industry to report, update, and analyze incidents for trend data

### Technologies Used
- ASP.NET
- HTML
- CSS
- Bootstrap
- C#
- JavaScript
- JQuery
